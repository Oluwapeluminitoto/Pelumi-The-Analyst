Flu cases Analysis

This project analyzes a simulated clinical database from a Rwandan clinic to model monthly trends in flu cases and treatment costs. Using SQL, it performs time-series aggregation to identify peak months for flu incidence and corresponding financial expenditures. The insights generated support the clinic’s financial planning team to forecast demand, allocate resources, and optimize budgeting for high-risk periods.

Adidas U.S. Sales Performance

This project provides a data-driven analysis of Adidas’s U.S. sales, identifying key performance indicators (KPIs), uncovering inefficiencies, and recommending strategic actions. By analyzing metrics like revenue, units sold, profit margin, channel performance (online, in-store, outlet), and regional trends, the work highlights areas for improvement. The end result? Four integrated, actionable strategies focused on optimizing inventory allocation, correcting channel imbalances, and refining pricing ultimately boosting profitability and building a more efficient, insight-led sales strategy.
